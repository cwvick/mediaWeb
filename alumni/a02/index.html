　活躍する卒業生＞ 赤堀このみさん


赤堀このみさん

 
グリー株式会社
事業統括本部　Japan Game事業本部　Web Game Studio部　Artグループ
テクニカルデザインチーム（アニメーション、映像制作部署）

現在は、どのようなお仕事をされていますか？

入社当時は、イラストを描く部署におりましたが、その後アニメーションの部署に異動してキャラクターモーションやエフェクト、たまにキャラクターデザインなどをしています。キャラクターは動かすことで命が吹き込まれるので、よりキャラクターの良さが引き出されます。そうやって自分が作ったキャラクターを見た人たちが「これはすごい」「かわいい」って言ってくれたりして、そんな時は、私がこいつらに命を吹き込んでやったんだぜって誇らしく思うんです。
初めのころは、アニメの知識が０の状態で、悩みながら仕事をしていましたが、まぁ何とかなるか精神で頑張ったら、社内でMVP取っていました。びっくりでしょ。私もびっくりした。何事もがむしゃらに頑張るといいことあるもんですね。

大学時代はどんなことを考えて、 どんなことに力を入れて勉強をしていましたか？

何も考えてなかったですね（笑）
でも、考えてなかったからこそ、自分のやりたいことは何でも全力でやりたいって思ってやってました。例え課題であろうが、作ったデザインはロフトとかハンズとかおしゃれなお店で売れるようなレベルにしたいな、とかそんな感じで、とりあえず誰かがほしくなるような作品を作りたい、と思いながら作ってました。そうするとだんだん妄想が膨らんで、頭の中がハイになってすべてが楽しく思えまして、やってやるぜー！って状態になれます。

女子美の後輩達や受験生に伝えたいメッセージをお聞かせ下さい。

まぁどんなこともなんとかなるさ！
ただ、なんとかなる！って思うだけじゃダメです。
なんとかするために、がむしゃらに動きましょう。
上手くいかないとき、チャンスが来ないとき、というか全然ダメなとき、そういう時は総じて、どっかで諦めたり、自分から動いていないことが原因です。
動いて何ぼ、やりのけて何ぼ、がむしゃらに走ってみて、たまに乗り越えられないでかい山にぶち当たって落ち込んでも、その時は一度休んでどうやったら登り切れるか考えてみると意外とふとした瞬間に答えがわかるもんです。
というわけで、なんとかするために、動いてみましょう。がんばれ。


　＜ 活躍する卒業生
